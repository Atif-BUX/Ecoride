## 🇬🇧 Project Overview (English Version)
**EcoRide** is a web application designed for **eco-friendly carpooling**, developed as part of the **Web & Mobile Web Developer ECF Project**.
Its goal is to provide a clean, simple interface to **search, filter, and view available rides**, while promoting **eco-friendly trips (electric vehicles)**.

### 🎯 Main Features
- 🏠 **Homepage**: company introduction and trip search fields.
- 🚗 **Carpool View (US 3)**: displays available trips by city and date.
- 🔍 **Filters & Sorting**: by price, duration, eco aspect, and driver rating.
- 👤 **Driver Profile**: name, photo, rating, and preferences.
- 💳 **Credit & Participation System** (US 6).
- 🧭 **Main Menu**: easy navigation between pages (Login / Contact / Home).
